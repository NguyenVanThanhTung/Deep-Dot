# DEEP-DOT
Artificial Intelligence midterm project

## 📋 Tổng quan dự án
Repository được tổ chức để theo dõi tiến độ qua các giai đoạn (projects):
* **Proj1:** Search
* **Proj2:** Multi-Agent Search
* **Proj3:** Reinforcement Learning
* **Proj4:** Ghostbusters (Bayesian Networks)
* **Proj5:** Machine Learning

## 🚀 Cách bắt đầu
1. **Clone repo:**
```bash
git clone [https://github.com/username/cs188-pacman-projects.git]
```
```bash
cd cs188-pacman-projects
```

2. **Cài đặt môi trường:** (Đảm bảo bạn đã cài đặt Python 3.x)
```bash
# Cài đặt các thư viện cần thiết nếu có
pip install -r requirements.txt
```

3. **Chạy bài tập:**
Để chạy code Pacman, ví dụ cho phần Search:
```bash
python pacman.py --layout testMaze --pacman GoWestAgent
```
